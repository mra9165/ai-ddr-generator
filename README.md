# AI DDR Generator

AI-powered Detailed Diagnostic Report generation system using:

- PDF text extraction
- Image extraction
- Thermal + visual evidence pairing
- Groq LLM reasoning
- PDF report generation
- Streamlit interface

## Features

- Upload inspection report PDF
- Upload thermal report PDF
- Generate structured DDR automatically
- Download final report

## Tech Stack

- Python
- PyMuPDF
- ReportLab
- Groq API
- Streamlit

## Run

pip install -r requirements.txt

streamlit run streamlit_app.py
